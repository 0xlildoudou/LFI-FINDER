## 📀 Installation 📀


```bash
git clone https://github.com/capture0x/LFI-FINDER/
cd LFI-FINDER
bash setup.sh
pip3 install -r requirements.txt
chmod -R 755 lfi.py
```

```bash
python3 lfi.py

```
